CSS Positioning



This article introduces the topic of laying out web pages
in CSS using a combination of posittioning schemes.

CSS alows you to use three different posittioning schemes to
create complex layouts:
  • Normal flow
  • Absolute posittioning
  • Floating

By using a combination of these schemes you do not need to resort to
using tables to lay out your pages.
